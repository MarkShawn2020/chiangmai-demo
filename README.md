## Features

- Interactive map of Chiang Mai golf courses
- Route planning between courses
- Course details: name, rating, distance
- Mobile-responsive design
- Bilingual support (English/Chinese)

## Demo

Visit the live demo: [chiangmai-demo.vercel.app](https://chiangmai-demo.vercel.app)

## Usage

Simply open `index.html` in a browser, or deploy to any static hosting service.

```bash
# Local development
npx serve .

# Deploy to Vercel
vercel
```

## Tech Stack

- **Leaflet.js** - Interactive maps
- **Leaflet Routing Machine** - Route planning
- **Vanilla HTML/CSS/JS** - No framework dependencies
- **Vercel** - Hosting
